`https://code.visualstudio.com/download`

###### Etapas de Instalação 
- ler e aceitar o acordo de licença
`Eu aceito o acordo`
- selecionar o local de destino
`C:\Users\User\AppData\Local\Programs\Microsoft VS Code`
- selecionar a pasta do menu iniciar
`Visual Studio Code`
- selecionar tarefas adicionais
`Criar um atalho para área de trabalho`
`Adicione a ação "Abrir com Code" ao menu de contexto de arquivo do Windows Explorer`
`Adicione a ação "Abrir com Code ao menu de contexto de diretório do Windows Explorer`
`Registre Code como um editor paa tipo de arquivos suportados`
`Adicione em PATH (disponível após reiniciaar)`
- pronto para instalar
- concluir
